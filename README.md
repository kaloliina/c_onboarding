# c_onboarding

## Project Overview

 This repository is a compilation of smaller C projects where the goal was to gradually learn new concepts along the way, starting all the way from the very basics of C programming! The purpose of these projects was to implement many standard library functions from scratch and gain a deeper understanding on how they actually work. In addition, there is get_next_line which is reading from a file descriptor line by line.

## Concepts Learned

- C basics: data types, memory management, pointers, strings
- Makefile
- Variadic Functions
- Conversion Processes
- File Handling
- Error Checking

## Learnings

- These projects were a great refresher for my C skills, covering all the basics I had learned (during application period to Hive Helsinki coding school) while also introducing more complex challenges. I gained a good understanding of C programming but I also learned the importance of putting attention to detail. My biggest takeaway for these projects was the importance of being meticulous. Since I was working with many standalone functions, I had to ensure each one worked perfectly. If I missed something, it could affect other functions and also be frustrating for future projects.

## How to Run the Project

- Clone the repository and navigate to the project you want to use
- For printf and libft, compile using make and link against the compiled library
- For get_next_line, add the source files directly to your project and include the header
